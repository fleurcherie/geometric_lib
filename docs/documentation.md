Документация проекта "Геометрические вычисления"

Общее описание
Проект содержит набор функций для вычисления площади и периметра простых геометрических фигур: круг, прямоугольник, треугольник.  
Каждая функция имеет docstring и примеры использования.


Функции проекта

Круг

 area(r)
Описание: Вычисляет площадь круга.  
Параметры: 
- `r (float)` — радиус круга.  
Возвращает:  
- `float` — площадь круга.  
Пример:
```python
area(3)  # 28.274333882308138

perimeter(r)

Описание: Вычисляет периметр (длину окружности) круга.
Параметры:

r (float) — радиус круга.
Возвращает:

float — периметр круга.
Пример:

perimeter(3)  # 18.84955592153876

Прямоугольник
rectangle_area(a, b)

Описание: Вычисляет площадь прямоугольника.
Параметры:

a (float) — длина прямоугольника

b (float) — ширина прямоугольника
Возвращает:

float — площадь прямоугольника
Пример:

rectangle_area(4, 5)  # 20

rectangle_perimeter(a, b)

Описание: Вычисляет периметр прямоугольника.
Параметры:

a (float) — длина прямоугольника

b (float) — ширина прямоугольника
Возвращает:

float — периметр прямоугольника
Пример:

rectangle_perimeter(4, 5)  # 18

Треугольник
triangle_area(a, h)

Описание: Вычисляет площадь треугольника по основанию и высоте.
Параметры:

a (float) — основание треугольника

h (float) — высота, проведённая к основанию a
Возвращает:

float — площадь треугольника
Пример:

triangle_area(6, 4)  # 12.0

triangle_perimeter(a, b, c)

Описание: Вычисляет периметр треугольника по трём сторонам.
Параметры:

a (float) — первая сторона

b (float) — вторая сторона

c (float) — третья сторона
Возвращает:

float — периметр треугольника
Пример:

triangle_perimeter(3, 4, 5)  # 12
